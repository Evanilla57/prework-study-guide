# <Prework Study Guide>

## Description

This project was designed to help boot camp students study HTML, CSS, Javascript and Git. It taught me a lot as the coder and I am happy to share what I hae learned with others.

## Usage

Please review the notes in each section to study each topic. You may open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows) for suggestions on what to focus on first. Within the console panel, there is a list of topics covered in the prework along with a suggestion on which topic to study first.
